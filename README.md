### Задание 1. Кеширование

Приведите примеры проблем, которые может решить кеширование.

Приведите ответ в свободной форме.

Кеширование позволяет решить следующие проблемы:

1. Высокая нагрузка: кэширование позволяет сократить количество запросов, снижая нагрузку на нее, что увеличивает производительность системы.
2. Медленная скорость: кэш в памяти позволяет получать данные значительно быстрее, чем из базы данных или других удаленных источников.
3. Несогласованность данных между серверами: использование внешнего кэша позволяет обеспечить согласованность данных между несколькими серверами приложения.
4. Проблемы с доступностью источника данных: если основной источник данных недоступен, кэш может предоставить последние доступные данные, предотвращая сбои в работе приложения.
5. Сглаживание пиковых нагрузок: кэширование позволяет справляться с резкими всплесками трафика, например, во время распродаж или других событий, когда нагрузка на систему многократно возрастает.
